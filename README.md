# 🛡️ Cyber-Shield Web Engine (v6.6.6)

**Cyber-Shield Web Engine** is a professional-grade credential auditing module built with the **Flask** framework. It offers a secure, responsive web interface designed to evaluate password entropy and brute-force resistance using multi-layered pattern matching algorithms.

---

## 🚀 Key Features
- **Modern Security Interface**: A sleek, dark-themed UI built for optimal user experience and clarity.
- **Secure Eye-Toggle**: Real-time visibility control for input credentials using Vanilla JavaScript.
- **Deep Entropy Analysis**: Automatically audits security metrics including string length, case diversity, numeric presence, and symbol density.
- **Engine Metadata Module**: Built-in system information module to track build version and developer credentials.

## 🛠️ Tech Stack
- **Backend:** Python 3.x / Flask Framework
- **Frontend:** HTML5, CSS3 (Modern Flexbox), JavaScript
- **Security Logic:** Regular Expression (Regex) based entropy mapping
